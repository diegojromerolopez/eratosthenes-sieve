# Eratosthenes' Sieve

An implemententation of the
[Sieve of Eratosthenes](https://en.wikipedia.org/wiki/Sieve_of_Eratosthenes)
in Golang.

# Use

Use eratosthenessieve.SieveOfErathostenes

```go
var primes []*big.Int
primes = SieveOfErathostenes(big.NewInt(33))
```
